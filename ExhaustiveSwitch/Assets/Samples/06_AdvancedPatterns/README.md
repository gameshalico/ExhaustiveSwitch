# 06_AdvancedPatterns - 高度なデザインパターン

このサンプルでは、ExhaustiveSwitchを活用した応用的なデザインパターンの実装例を示します。

## 概要

以下のパターンが含まれています:

1. **ステートマシン**
   - 状態遷移を型安全に実装
   - すべての状態を確実に処理
   - 状態ごとに異なる振る舞い

2. **コマンドパターン**
   - 操作の履歴管理
   - Undo/Redo機能
   - トランザクション処理

3. **式木パターン**
   - データと処理の分離
   - ダブルディスパッチが不要
   - 式の評価、最適化、表示など複数の処理を追加可能

## 各パターンの使い所

### ステートマシン
- AIの行動制御
- ゲームの状態管理（タイトル画面、ゲーム中、ポーズなど）
- アニメーション制御

### コマンドパターン
- パズルゲームの1手戻し機能
- エディタのUndo/Redo
- ネットワークゲームのコマンド送信
- マクロ機能

### 式木パターン
- 数式の評価と計算
- コンパイラやインタプリタの実装
- 式の最適化（定数畳み込みなど）
- クエリビルダー
